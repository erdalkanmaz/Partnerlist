Excel-Format

Ihre Excel-Datei sollte folgende Spalten (Überschriften) enthalten. Groß-/Kleinschreibung ist egal:

  • Partner/Agent (Unternehmensname) - PFLICHT
  • Web (Webadresse) - PFLICHT
  • e-mail (E-Mail) - PFLICHT
  • Adress (Adresse) - PFLICHT
  • Person (Personalname) - Optional
  • Gender, Title, Telephone, Fax - Optional
  • Umsatzsteuer-ID, Amtsgericht, Kommentar - Optional
